```
cd fitMart
npm install
npm start
```

## **Key Features**

- Homepage
- Video listing page
- Filters by Category
- Playlist Management
- Like/ Dislike
- Watch Later
- History
- Authentication
  - Login
  - Logout
  - Signup
- Single Video Page
- Loading and Alerts
- User Profile Page

## **Tech Stack**

- ReactJS
- CSS
- [Mockbee](https://mockbee.netlify.app/) - Mock Backend
- [Colorant UI](https://colorant-ui.netlify.app) - Component Library
